# VetSphere Pet Care Management System 🐾

Welcome to **VetSphere**, a comprehensive Django-based application designed to streamline pet care services, clinic management, and customer relationship management (CRM) for veterinary clinics and pet care centers.

## Features ✨
- **Pet Management**: Easily register and manage pet details, including medical history and daily logs.
- **Booking System**: Schedule and manage appointments for various pet care services.
- **CRM Dashboard**: Keep track of user interactions, messages, and overall clinic activities.
- **Billing & Invoicing**: Generate invoices and manage payments seamlessly.
- **Staff Rosters**: Organize and manage staff schedules efficiently.
- **Secure Authentication**: Robust login and registration system for admins and users.

## Technologies Used 🛠️
- **Backend**: Python, Django
- **Database**: SQLite (Default)
- **Frontend**: HTML, CSS, JavaScript
- **Version Control**: Git & GitHub

## How to Run Locally 🚀
1. Clone the repository: `git clone https://github.com/SELVAKUMAR-ANALYST/VetSphere.git`
2. Navigate to the project folder: `cd VetSphere`
3. Activate the virtual environment (if using one).
4. Install dependencies: `pip install -r requirements.txt`
5. Run the server: `python manage.py runserver`
6. Open your browser and go to `http://127.0.0.1:8000/`
